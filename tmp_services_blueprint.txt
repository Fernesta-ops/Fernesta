■  On-Page SEO
1
Keyword Research
Find words people type on Google to discover businesses
like yours.
2
Title Tag Writing
Write the blue clickable headline shown in Google
search results.
3
Meta Description
Write the short summary shown below the title in search
results.
4
Header Tags (H1–H3)
Organize page content with headings like chapters in a
book.
5
Content Optimization
Improve page text to naturally include target keywords.
6
Image Alt Text
Add text descriptions to every image so Google them.
understands
7
Internal Linking
Link related pages together so users and Google easily.
navigate
8
URL Structure
Make URLs short and keyword-rich (e.g. /seo-services
not /page?id=4).
■  Off-Page SEO
1
Backlink Building
Get other websites to link to ours — like votes of for
trust Google.
2
Guest Posting
Write articles on other blogs in exchange for a link to
back the site.
3
Directory Submissions
List the business on Justdial, Sulekha, Google Business
etc.
4
Social Bookmarking
Share pages on Mix, Tumblr, Reddit to create extra
entry points.
5
Forum & Q&A Replies
Answer questions on Quora and forums with a natural
site link.
6
Competitor Link Audit
Study competitor backlinks with Ahrefs to target same
sources.
7
Bad Link Removal
Find and disavow spammy backlinks that can hurt
rankings.
8
PR & Brand Mentions
Get journalists or bloggers to mention and link to the
brand.
■■  Technical SEO
1
Page Speed Optimization
Compress images and clean code so pages load fast.
2
Mobile Responsiveness
Ensure the site looks perfect on phones and tablets.
3
XML Sitemap
Submit a map of all pages to Google so it indexes them
properly.
4
Robots.txt Setup
Tell Google which pages to crawl and which to ignore.
5
SSL / HTTPS
Ensure the site has a security certificate (the green
padlock).
6
Broken Link Fixing
Find and repair links that lead to error / 404 pages.
7
Core Web Vitals
Fix loading, interactivity & visual stability scores in
Search Console.
8
Schema Markup
Add hidden code so Google shows star ratings, FAQs in
results.
